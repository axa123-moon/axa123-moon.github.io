df = pd.read_csv(path_house)

2. Null 확인 방법

2.1 Null 확인하기

  • DataFrame 전체 Null 확인하기
    df.isnull().sum()

  • 한개 column Null 확인하기
    col = [‘column’]
    df[col].isnull().sum()

  • 모든 column에 대한 Null 비율 확인하기
    for col in df.columns:
    msg = ‘column: {:>10}\t Percent of NaN value: {:.2f}%’.format(col, 100 * (df_train[col].isnull().sum() / df_train[col].shape[0]))
    print(msg)

  • 특정 column에 Null이 있는 Null 확인하기
    df[df[col].isnull()]

3. Column별 내용 확인하기

col = [‘column’]

  • 한개 column 내용중에 unique() 확인하기
    df[col].unique()

  • 한개 column 내용중에 데이터 고유값 확인하기
    df[col].nunique()

  • 한개 column 내용 확인하기
    df[col].value_counts()

  • 모든 column 내용중에 데이터 고유값 확인하기
    for col in df_train.columns:
    msg = ‘column: {}, unique kind: {}’.format(col, df_train[col].nunique())
    print(msg)

4. 불필요한 row 또는 column 삭제하기 (Delete)

  • column 삭제하기
    cols = [‘column1’, ‘column2’]
    df.drop(cols)

  • 특정 row 삭제하기
    col = [‘column’]
    cond = df[col] == 0
    df = df.drop(df[cond].index)

  • 데이터가 있지 않은 col 정리
    df = df.dropna(axis = 1, how = ‘all’)

  • 결측치 전부 제거. row에 하나라도 null이 있으면 해당 row 삭제
    df = df.dropna(axis = 0, how = ‘any’)
    df.info()

5. 채우기 (Fill)

col = [‘column’]

  • df[col] 값이 null일 경우 0으로 채우기
    df[col] = df[col].fillna(0)

  • df[col] 값이 null일 경우 ‘column’의 평균값으로 채우기
    df[col] = df[col].fillna(df[col].mean())

  • df[col] 값이 null일 경우 ‘column’의 중앙값으로 채우기
    df[col] = df[col].fillna(df[col].median())

  • df[col] 값이 null일 경우 ‘column’의 최소값으로 채우기
    df[col] = df[col].fillna(df[col].min())

  • df[col] 값이 null일 경우 ‘column’의 최대값으로 채우기
    df[col] = df[col].fillna(df[col].max())

  • df[col] 값이 null일 경우 ‘column’의 최빈값으로 채우기 (많이 사용한 값)
    df[col] = df[col].fillna(df[col].mode()[0])

  • df[col] 값이 null일 경우 특정 컬럼(Age)을 기준으로 평균값 할당
    df[col] = df[col].fillna(df.groupby(‘Age’)[‘column’].transform(‘mean’))

  • df[col] 값이 null일 경우 특정 컬럼(Age)을 기준으로 평균값 할당
    df[col] = df[col].fillna(df.groupby(‘Age’)[‘column’].transform(‘median’))

  • 앞 값으로 채우기 (시계열 데이터 채울때)
    df[col] = df[col].fillna(method=’ffill’)

  • 뒷 값으로 채우기 (시계열 데이터 채울때)
    df[col] = df[col].fillna(method=’bfill’)

  • 시계열데이터에서 선형으로 비례하는 방식으로 결측값 보간
    df = df.interpolate()

7. 카테고리 변수 처리하기 (Categorical Features)

  • Feature의 데이터 type을 category로 변경하기

7.1 라벨 인코딩 (Label encoding)

  • Feature의 내용을 문자에서 숫자로 변환시킴. 모델에 학습에 사용하기 위함.

display(df[cols].head(1)) # 변경후

7.4 LabelCount encoding (랭킹)

  • Feature에 대한 랭킹을 계산해줌

def labelcount_encode(X, categorical_features, ascending=False):
print(‘LabelCount encoding: {}’.format(categorical_features))
X_ = pd.DataFrame()
for cat_feature in categorical_features:
cat_feature_value_counts = X[cat_feature].value_counts()
value_counts_list = cat_feature_value_counts.index.tolist()
if ascending:
# for ascending ordering
value_counts_range = list(
reversed(range(len(cat_feature_value_counts))))
else:
# for descending ordering
value_counts_range = list(range(len(cat_feature_value_counts)))
labelcount_dict = dict(zip(value_counts_list, value_counts_range))
X_[cat_feature] = X[cat_feature].map(
labelcount_dict)
X_ = X_.add_suffix(‘labelcount_encoded’)
if ascending:
X = X_.add_suffix(‘ascending’)
else:
X = X_.add_suffix(‘descending’)
X = X_.astype(np.uint32)
return X_

df[‘LotArea’] = labelcount_encode(df, [‘LotArea’])
df.head(3)

df[‘LotAreaRank’] = df[‘LotArea’].rank(method=’min’, ascending=False)와 동일함

8. 수치 변수 처리하기 (Numerical Features)

8.1 Standard Scaling

  • 평균을 0, 분산을 1로 변경

from sklearn.preprocessing import StandardScaler
data = [[0, 0], [0, 0], [1, 1], [1, 1]]
scaler = StandardScaler()
print(scaler.fit(data))
print(scaler.mean_)
print(scaler.transform(data))
print(scaler.transform([[2, 2]]))

8.2 MinMax Scaling

  • MinMax Scaling 0과 1사이로 변경
    from sklearn.preprocessing import MinMaxScaler
    data = [[-1, 2], [-0.5, 6], [0, 10], [1, 18]]
    scaler = MinMaxScaler()
    print(scaler.fit(data))
    print(scaler.data_max_)
    print(scaler.transform(data))
    print(scaler.transform([[2, 2]]))

8.3 Nomalization

  • 정규화시킴

from sklearn.preprocessing import Normalizer
X = [[4, 1, 2, 2],
[1, 3, 9, 3],
[5, 7, 5, 1]]
transformer = Normalizer().fit(X) # fit does nothing.
transformer
transformer.transform(X)

8.4 RobustScaler

  • Standard와 유사 하나 평균과 분산 대신, median과 quartile을 사용

[ -2., 1., 3.],
[ 4., 1., -2.]]

transformer
transformer.transform(X)
