EDA 진행관련 자주 사용하는 python code를 기록하였습니다.

1. 데이터의 기본 정보 보기

  • DataFrame 데이터 보기
    df.head()

  • DataFrame의 info 보기
    df.info()

  • DataFrame의 description 보기
    df.describe() # type이 int와 float 정보 보여줌
    df.describe(include=[‘O’]) # type이 object 정보 보여줌

2. Null 확인 방법

2.1 Null 확인하기

  • DataFrame 전체 Null 확인하기
    df.isnull().sum()

  • 한개 column Null 확인하기
    df[‘column’].isnull().sum()

  • 모든 column에 대한 Null 비율 확인하기
    for col in df.columns:
    msg = ‘column: {:>10}\t Percent of NaN value: {:.2f}%’.format(col, 100 * (df_train[col].isnull().sum() / df_train[col].shape[0]))
    print(msg)

  • 특정 column에 Null이 있는 Null 확인하기
    df[df[‘column’].isnull()]

3. Column별 내용 확인하기

  • 한개 column 내용 unique() 확인하기
    df[‘column’].unique()

  • 한개 column 내용중에 데이터 고유값 확인하기
    df[‘column’].nunique()

  • 한개 column 내용 확인하기
    df[‘column’].value_counts()

  • 모든 column 내용중에 데이터 고유값 확인하기
    for col in df_train.columns:
    msg = ‘column: {}, unique kind: {}’.format(col, df_train[col].nunique())
    print(msg)

4. 불필요한 row 또는 column 삭제하기

  • column 삭제하기
    del_col_list = [‘column1’, ‘column2’]
    df.drop(col_list)

  • 특정 row 삭제하기
    cond = df[‘column1’] == 0
    df = df.drop(df[cond].index)

  • 데이터가 있지 않은 col 정리
    df = df.dropna(axis = 1, how = ‘all’)

  • 결측치 전부 제거. row에 하나라도 null이 있으면 해당 row 삭제
    df = df.dropna(axis = 0, how = ‘any’)
    df.info()

5. 채우기

  • df[‘column’] 값이 null일 경우 ‘column’의 평균값으로 채우기
    df[‘column’] = df.fillna(df[‘column’].mean())
