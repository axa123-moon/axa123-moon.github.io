블로그 제작하기 위한 환경 설정

Posted on 2023-11-21 Edited on 2023-11-22

안녕하세요. 저의 블로그에 오신것을 환영합니다.

블로그 제작 관련 내용입니다.

설치관련 순서는 아래와 같습니다.

  1. nodejs 설치

  2. hexo 설치

  3. hexo로 블로그 작성 및 생성
    hexo new post “My first post”
    hexo clean && hexo generate

  4. github repository 생성

  5. hexo 환경설정
    _config.yml 파일 수정 및 저장

  6. local서버에서 배포 후 확인
    hexo server

  7. github 배포
    hexo deploy

  8. theme 교체

  9. github 재배포

hexo를 사용하여 제작한 blog를 github에 배포하는 동영상 정보를 공유합니다.
